# ROADMAP v2 — Direct merchant API (POST /v1/bills)

**You are done with v2 when** an API-key-authenticated `POST /v1/bills` carrying line items at two different tax rates writes **exactly one** Order + N OrderItems + Bill + Link + PENDING Broadcast in **one transaction** and returns `201 {bill_id, identifier, url}`; **replaying the same `external_transaction_id` returns `200` and changes nothing**; a one-paise total mismatch or a missing HSN returns `422` with **zero rows written**; `jbill.in/<identifier>` renders the itemized invoice with a GST breakdown whose per-line taxes sum exactly to the printed total; the unchanged drainer delivers it to Mailhog — with **no floats in the money path**, **no PII in logs**, and **every v1 test still green**.
